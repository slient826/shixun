HTML5简介

2018-08-08 by shixun

HTML5 是 HTML4 的一个升级版本，简称 H5，标准通用标记语言下的一个应用HTML标准自1999年12月发布的HTML4.01后，后继的HTML5和其它标准被束之高阁，为了推动Web标准化运动的发展，一些公司联合起来，成立了一个叫做 Web Hypertext Application Technology Working Group （Web超文本应用技术工作组 -WHATWG） 的组织。WHATWG 致力于 Web 表单和应用程序，而W3C（World Wide Web Consortium，万维网联盟） 专注于XHTML2.0。在 2006 年，双方决定进行合作，来创建一个新版本的 HTML。

HTML5新增特性

  • 新的语义标签 header/footer/article/section...
  • 新增表单控件（标签）data、time、email、url、number…
  • 新增视频/音频标签video、audio
  • 新增画布标签（canvas）
  • 更好支持支持本地离线数据存储
